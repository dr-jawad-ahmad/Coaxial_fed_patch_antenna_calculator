Equations reference (Balanis, Antenna Theory: Analysis and Design, 3rd ed., Ch. 14)

Eq. Expression Description
14-37 W = c / (2fᵣ) · √(2 / (εᵣ+1)) Optimal patch width for maximum radiation
14-1 εᵣₑff = (εᵣ+1)/2 + (εᵣ−1)/2 · (1+12h/W)^(−½) Effective permittivity — Hammerstad formula
14-2 ΔL/h = 0.412 · (εᵣₑff+0.3)(W/h+0.264) / [(εᵣₑff−0.258)(W/h+0.8)] Length extension due to fringing fields
14-38/39 L = c / (2fᵣ√εᵣₑff) − 2ΔL Actual resonant patch length
14-3 fᵣ = c / [2(L+2ΔL)√εᵣₑff] Actual resonant frequency (verification)
14-44 G₁ = (1/120π²) ∫₀^π [sin(k₀W cosθ/2) / cosθ]² sin³θ dθ Single-slot radiation conductance (numerical)
14-47 G₁₂ = (1/120π²) ∫₀^π [sin(k₀W cosθ/2) / cosθ]² J₀(k₀L sinθ) sin³θ dθ Mutual conductance between radiating slots (numerical, uses J₀)
14-50 Rᵢₙ = 1 / [2(G₁ + G₁₂)] Input resistance at radiating edge — even TM₁₀ mode
14-60 Rᵢₙ(y₀) = Rᵢₙ · cos²(πy₀/L) → y₀ = L/π · arccos(√(Z₀/Rᵢₙ)) Coaxial feed inset position for Z₀ impedance match
14-63 BW ≈ 3.77 · (εᵣ−1)/εᵣ² · (W/L) · (h/λ₀) [× 100%] Fractional bandwidth (VSWR ≤ 2:1, thin substrate approx.)
— D₀ = W² / (30π λ₀² G₁) Broadside directivity from two-slot model
Balanis, C. A. — Antenna Theory: Analysis and Design, 3rd ed., Wiley, 2005. Numerical integration: Simpson's rule (N=2000). Bessel J₀: Abramowitz & Stegun polynomial.
