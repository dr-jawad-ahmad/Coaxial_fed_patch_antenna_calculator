# Coaxial_fed_patch_antenna_calculator
Rectangular microstrip patch antenna calculator based on Balanis *Antenna Theory* Ch. 14. Computes patch dimensions (W, L, εᵣeff, ΔL), radiation conductances G₁ &amp; G₁₂ via numerical integration, edge resistance Rᵢₙ, coaxial feed inset y₀ for 50 Ω match, bandwidth, and directivity. Single HTML file — no dependencies.
